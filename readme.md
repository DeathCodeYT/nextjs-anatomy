# Next.js Anatomy 🚀

**Next.js ko sirf use karna nahi, balki andar se samajhna.**

Ye ek **course-first roadmap** hai, project-first nahi.
Projects yahan sirf isliye hain taaki jo concepts aap seekhein, unko real world me apply kar paayein — concepts se focus kabhi nahi hatega.

---

## 🎯 Ye Course Kis Baare Mein Hai

Aaj ke time par zyadaatar Next.js courses sirf *kya likhna hai* ye sikha dete hain.

**Next.js Anatomy aapko ye sikhata hai:**

* **Why** – koi feature exist kyu karta hai
* **How** – wo internally kaise kaam karta hai
* **When** – kis situation me wo break ho sakta hai
* **Where** – kahan use karna chahiye aur kahan bilkul nahi

Course ke end tak aap tutorials pe depend nahi rahenge —
aap khud logic se problems solve kar paayenge.

---

## 🧠 Ye Course Kis Ke Liye Hai

Aapko pehle se ye aana chahiye:

* JavaScript (async, promises, closures)
* React (hooks, state, basic architecture)
* HTML, CSS, Tailwind
* Kam se kam 1–2 React projects bana chuke ho

❌ Ye beginner crash course nahi hai
✅ Ye **mastery-oriented course** hai

---

## 🧩 Course Ka Structure Kaise Hoga

* **Ek topic = ek video**
* Topics bilkul **level-wise** arrange honge, random nahi
* Frontend aur backend ko **alag-alag samjhaya jaayega**, lekin story connected rahegi
* Debugging hamesha **usi jagah sikhaayi jaayegi jahan wo hoti hai** (frontend vs backend)
* Edge Runtime, Node Runtime aur VPS hosting **jab zaroori hoga tab clearly discuss honge**

Har stage par aapko clear rahega:

> *Is situation me Edge pe kya hoga? Node pe kya hoga? VPS par kya hoga?*
